Input box in the grid header

Create a new AW.UI.Input control and put it into the header with setHeaderTemplate(obj, col) method

	var input = new AW.UI.Input;
	grid.setHeaderTemplate(input, 0);

Extended grid allows multiple header rows. Use setHeaderTemplate(obj, col, row) method

	grid.setHeaderTemplate(input1, 0, 1); // first column, second header row
	grid.setHeaderTemplate(input2, 1, 1); // second column, second header row
