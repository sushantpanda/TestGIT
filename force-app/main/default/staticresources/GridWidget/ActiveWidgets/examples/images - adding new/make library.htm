How to create icon library file.

Paste the list of image filenames into the column A of this Excel spreadsheet (icons.xls)

Copy the html text from the column B to the 'images' div below this paragraf (see the source).

Make a screen capture of this page, extract the image, set transparent color and save it as 8-bit (255 colors) png.

At the end the image should look like this one.

Enter the image filename into the cell I3 and save the CSS rules in the column J to the new stylesheet file (.css).

The CSS file will contain the links to all images in the library.

Copy both .png and .css files to your resource directory and add .css link to the page.

The final result: icon library image, css file.
